The `frontend` branch of this repo is synced to the front_end folder of the main repo at https://capstone.cs.utah.edu/kesslersimdrome/kesslersimdrome.

### How to sync the deployed page:
1. Actions -> Sync frontend from GitLab -> Run workflow
2. Wait a few minutes for `Sync frontend from GitLab` and `pages-build-deployment` workflows to run
3. Settings -> Pages -> Custom Domain -> Enter `kesslersimdrome.org` -> Save
4. Wait a few minutes for changes to be fully applied
